テクノロジ系 アルゴリズム（ハッシュ法） ハッシュ関数ハッシュ表衝突シノニムmod演算

問題文

自然数をキーとするデータを，ハッシュ表を用いて管理する。キーxのハッシュ関数h(x)を

h(x) = x mod n

とすると，任意のキーaとbが衝突する条件はどれか。ここで，nはハッシュ表の大きさであり，x mod nはxをnで割った余りを表す。

ア　a+bがnの倍数　　　　イ　a−bがnの倍数
ウ　nがa+bの倍数　　　　エ　nがa−bの倍数

正解

イ：a−bがnの倍数

選択肢補足

選択肢 内容 補足
ア a+bがnの倍数 a, bを n進数表現（商と余り）で表して検証すると、衝突条件とは一致しない。反例（n=3, a=4, b=5など）で容易に崩れる
イ a−bがnの倍数 正解。a = nα + r, b = nβ + r（rは共通の余り）とおくと a − b = n(α − β) となり、a−bは必ずnの倍数になる。逆にa−bがnの倍数であれば、aとbをnで割った余りは一致し、衝突する
ウ nがa+bの倍数 「nがa+bの倍数」はa+bがnを割り切ることを意味し、衝突条件とは無関係。反例で容易に崩れる
エ nがa−bの倍数 「nがa−bの倍数」はa−bがnを割り切ることを意味し、正解の「a−bがnの倍数（nがa−bを割り切る）」とは主従が逆になっている誤り

解き方

  1. 問題文のキーワード・条件を整理する。
  2. ハッシュ関数 h(x) = x mod n において、異なるキーa, bが同じハッシュ値を持つ（衝突する）条件を求める。
  3. 衝突の定義を数式で表す。
  4. 衝突するとは h(a) = h(b)、すなわち a mod n = b mod n が成り立つことである。
  5. a, bをnで割った商と余りで表現する。
  6. 自然数α, β（商）と共通の余りrを用いて、a = nα + r、b = nβ + r と表せる（h(a)=h(b)=rのとき）。
  7. a − b を計算する。
  8. a − b = (nα + r) − (nβ + r) = n(α − β)
  9. これは「nに整数(α−β)を掛けた値」であり、a−bは必ずnの倍数になる。
  10. 計算による網羅的検証（複数のn, a, bの組み合わせで衝突の有無と各選択肢の条件を比較）でも、「a−bがnの倍数」であることと「衝突する」ことが完全に同値であることを確認した（他の選択肢ア・ウ・エはいずれも反例が存在し、衝突条件と一致しない）。
  11. 以上より、イ（a−bがnの倍数）を正解と判断する。
