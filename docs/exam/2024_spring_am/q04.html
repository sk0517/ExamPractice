テクノロジ系 基礎理論（情報に関する理論・誤り訂正符号） ハミング符号誤り訂正mod2演算パリティ検査

問題文

符号長7ビット，情報ビット数4ビットのハミング符号による誤り訂正の方法を，次のとおりとする。

受信した7ビットの符号語 x1 x2 x3 x4 x5 x6 x7（xkは0又は1）に対して

c0 = x1 + x3 + x5 + x7
c1 =      x2 + x3      + x6 + x7
c2 =                x4 + x5 + x6 + x7

（いずれも mod 2 での計算）

を計算し，c0，c1，c2 の中に少なくとも一つは0でないものがある場合には，

i = c0 + c1×2 + c2×4

を求めて，左から i ビット目を反転することによって誤りを訂正する。

受信した符号語が 1000101 であった場合，誤り訂正後の符号語はどれか。

ア　1000001　　イ　1000101　　ウ　1001101　　エ　1010101

正解

エ：1010101

選択肢補足

選択肢 内容 補足
ア 1000001 計算結果のi=3と異なる位置（7ビット目相当）を反転した値であり、誤り訂正手順と一致しない
イ 1000101 受信した符号語そのままであり、誤り訂正処理を行っていない（c0・c1が0でないため訂正が必要）
ウ 1001101 反転すべき位置（3ビット目）とは異なる位置（4ビット目相当）を反転した値であり、計算結果と一致しない
エ 1010101 正解。c0=1, c1=1, c2=0 より i=1+1×2+0×4=3 となり、左から3ビット目（x3）を反転すると 1000101 → 1010101 となる

解き方

  1. 受信符号語をビットごとに分解する。
  2. x1=1, x2=0, x3=0, x4=0, x5=1, x6=0, x7=1（bash_toolでも同様に変数化して検証）。
  3. c0, c1, c2 をそれぞれ mod 2 で計算する。
  4. c0 = x1+x3+x5+x7 = 1+0+1+1 = 3 → mod 2 で 1
  5. c1 = x2+x3+x6+x7 = 0+0+0+1 = 1 → mod 2 で 1
  6. c2 = x4+x5+x6+x7 = 0+1+0+1 = 2 → mod 2 で 0
  7. c0, c1, c2 のいずれかが0でないことを確認し、誤りがあると判断する。
  8. c0=1, c1=1 が0でないため、誤り訂正処理が必要となる。
  9. 反転すべきビット位置 i を計算する。
  10. i = c0 + c1×2 + c2×4 = 1 + 1×2 + 0×4 = 3。bash_toolでの実計算でも i=3 を確認済み。
  11. 左から3ビット目（x3）を反転する。
  12. 受信符号語 1000101 の3ビット目（0）を反転すると 1010101 となる。
  13. 計算検証（Pythonでの実装結果）と照合し、訂正後の符号語が1010101と一致するエを正解と判断する。
