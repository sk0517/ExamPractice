## 問題文

ある組織ではCSIRTとPSIRTが設置されている。セキュリティインシデントのうち，PSIRTが対応するものはどれか。

ア　組織が顧客に販売した製品とその利用において発生したセキュリティインシデント
イ　組織が製品の在庫を管理するシステムにおいて発生したセキュリティインシデント
ウ　組織が製品の販売を管理するシステムにおいて発生したセキュリティインシデント
エ　組織の従業員の個人情報を管理するシステムにおいて発生したセキュリティインシデント

（なし）

## 正解

**ア**：組織が顧客に販売した製品とその利用において発生したセキュリティインシデント

## 選択肢補足

| 選択肢 | 内容 | 補足 |
|:--|:--|:--|
| **ア** | **組織が顧客に販売した製品とその利用において発生したセキュリティインシデント** | **正解。PSIRTは企業が製造・販売する「製品」そのものに関するセキュリティインシデントに対応する組織であるため、これに該当する** |
| イ | 組織が製品の在庫を管理するシステムにおいて発生したセキュリティインシデント | 在庫管理システムは組織内部の情報システムであり、これに関するインシデントはCSIRTの対応範囲 |
| ウ | 組織が製品の販売を管理するシステムにおいて発生したセキュリティインシデント | 販売管理システムも組織内部の情報システムであり、これに関するインシデントはCSIRTの対応範囲 |
| エ | 組織の従業員の個人情報を管理するシステムにおいて発生したセキュリティインシデント | 従業員情報を管理するシステムも組織内部の情報システムであり、これに関するインシデントはCSIRTの対応範囲 |

## 解き方

1. PSIRTとCSIRTの定義・対象範囲の違いを確認する。
   - PSIRT（Product Security Incident Response Team）：企業が製造・開発・販売する「製品やサービス」に関するセキュリティの向上、インシデント対応を行う組織。
   - CSIRT（Computer Security Incident Response Team）：企業・組織内部の「情報システムやネットワーク」など情報資産に関するセキュリティインシデント対応を行う組織。
2. ポイントとなる切り口を明確にする。
   - 「対象が"製品そのもの（顧客に提供されるもの）"か」「対象が"組織内部のシステム（業務で使う社内システム）"か」で、PSIRTとCSIRTのどちらが担当するかが分かれる。
3. 各選択肢を、この切り口で分類する。
   - ア：「顧客に販売した製品とその利用」における問題 → 製品そのものに関するインシデント → PSIRTの対応範囲。
   - イ：「製品の在庫を管理するシステム」における問題 → 社内の業務システム（情報システム）に関するインシデント → CSIRTの対応範囲。
   - ウ：「製品の販売を管理するシステム」における問題 → 同じく社内の業務システムに関するインシデント → CSIRTの対応範囲。
   - エ：「従業員の個人情報を管理するシステム」における問題 → 社内の情報システム（人事系システムなど）に関するインシデント → CSIRTの対応範囲。
4. 「製品そのもの」に関するインシデントである**ア**のみがPSIRTの対応範囲に該当すると判断する。
