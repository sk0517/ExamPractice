# 応用情報技術者試験 過去問サイト

応用情報技術者試験（AP）の過去問をブラウザで閲覧するための静的サイトです。

**公開URL：** https://sk0517.github.io/ExamPractice/

## 特徴

- 問題文を表示した状態で最初は解答・解説を非表示
- 「答えを見る」ボタンで正解・選択肢補足・解き方を開示
- PC・スマートフォン両対応

## 収録問題

| 試験回 | 問題数 |
|:--|:--|
| 2025年度 秋期 午前 | 80問 |
| 2025年度 春期 午前 | 80問 |
| 2024年度 秋期 午前 | 80問 |

## ディレクトリ構成

```
ExamPractice/
├── exam/                  # 問題ソース（Markdownファイル + 画像）
│   ├── 2025_autumn_am/
│   ├── 2025_spring_am/
│   └── 2024_autumn_am/
├── docs/                  # 生成済みHTML（GitHub Pages 公開ディレクトリ）
├── build.py               # 静的サイトジェネレーター
└── README.md
```

## ローカルでのビルド方法

```bash
# 依存パッケージのインストール（初回のみ）
pip install markdown

# HTMLの生成
python build.py
```

`docs/` に HTML ファイルが生成されます。

## 問題を追加するとき

1. `exam/<試験回>/` に Markdown ファイルを作成（フォーマットは既存ファイルを参照）
2. `python build.py` を実行
3. `docs/` をコミット・プッシュ

## 著作権

過去問の著作権は IPA（独立行政法人 情報処理推進機構）に帰属します。
本サイトは教育目的での利用であり、IPA の定める利用条件に従って掲載しています。
