## 問題文

fact(n) は，非負の整数 n に対して n の階乗を返す。fact(n) の再帰的な定義はどれか。

ア　if n=0 then return 0 else return n×fact(n−1)
イ　if n=0 then return 0 else return n×fact(n+1)
ウ　if n=0 then return 1 else return n×fact(n−1)
エ　if n=0 then return 1 else return n×fact(n+1)

## 正解

**ウ**：if n=0 then return 1 else return n×fact(n−1)

## 選択肢補足

| 選択肢 | 内容 | 補足 |
|:--|:--|:--|
| ア | 基底ケースで0を返す，n−1で再帰 | n=0のとき0!は本来1であるべきだが0を返しており、結果として全ての階乗計算結果が常に0になってしまうため誤り |
| イ | 基底ケースで0を返す，n+1で再帰 | 基底ケースの誤りに加え、n+1方向への再帰は再帰呼び出しがn=0に到達せず無限に増加し続け、停止しないため誤り |
| **ウ** | **基底ケースで1を返す，n−1で再帰** | **正解。0!=1という正しい基底ケースを満たし、再帰呼び出しがn−1方向に進むことで必ずn=0に到達し停止する** |
| エ | 基底ケースで1を返す，n+1で再帰 | 基底ケースは正しいが、n+1方向への再帰はn=0に到達せず無限に増加し続け、停止しないため誤り |

## 解き方

1. 階乗の数学的定義を確認する。
   - n! は「0! = 1」を基底ケースとし、n≧1のときは「n! = n×(n−1)!」という漸化式で定義される。
2. 各選択肢の基底ケース（n=0のときの戻り値）を確認する。
   - ア・イは「return 0」となっており、0!=1という正しい定義と矛盾するため、この時点で誤りと判断できる。
   - ウ・エは「return 1」となっており、正しい基底ケースと一致する。
3. ウ・エについて再帰呼び出しの方向を確認する。
   - 再帰が正しく停止するためには、呼び出しのたびに引数がn=0（基底ケース）に近づいていく必要がある。
   - ウは「n×fact(n−1)」であり、呼び出すたびに引数が1ずつ減少し、最終的にn=0に到達して停止する。
   - エは「n×fact(n+1)」であり、呼び出すたびに引数が増加し続けるため、n=0に到達せず無限再帰に陥り、決して停止しない。
4. 以上より、基底ケースと再帰呼び出しの方向の両方が正しい**ウ**を正解と判断する。
