テクノロジ系 ヒューマンインタフェース（アイコン設計規格） JIS X 9303-1アイコン習得性ユーザインタフェース

問題文

JIS X 9303-1:2006（ユーザシステムインタフェース及びシンボル―アイコン及び機能―アイコン一般）で規定されているアイコンの習得性の説明はどれか。

ア　アイコンによって表現されたシステム機能が、それが理解された後に、どれだけ容易に思い出すことができるかを示す。
イ　アイコンの図柄の詳細を、どれだけ容易に区別できるかを示す。
ウ　同じ又は類似したアイコンによる以前の経験に基づいて、どれだけ容易にアイコンを識別できるかを示す。
エ　空間的、時間的又は文脈的に近くに表示された別のアイコンから、与えられたアイコンをどれだけ容易に区別できるかを示す。

正解

ア：アイコンによって表現されたシステム機能が、それが理解された後に、どれだけ容易に思い出すことができるかを示す。

選択肢補足

選択肢 内容 補足
ア 理解後にどれだけ容易に思い出せるか 正解。習得性（Learnability）は、ユーザーが一度理解したアイコンの意味を、時間が経過した後でもどれだけ容易に思い出し、再認識できるかを表す特性
イ 図柄の詳細をどれだけ容易に区別できるか これは判別性・識別のしやすさに関する記述であり、習得性（記憶からの想起しやすさ）とは異なる観点
ウ 以前の経験に基づいてどれだけ容易に識別できるか これは既知性（familiarity）に近い説明であり、規格上の「習得性」の定義とは異なる
エ 近くに表示された別のアイコンからどれだけ容易に区別できるか これは規格で定義される判別性（discriminability：空間的・時間的・文脈的に近接した別のアイコンとの区別のしやすさ）の説明であり、習得性とは別の特性を指す

解き方

  1. 問題文のキーワードを整理する。
  2. 「JIS X 9303-1:2006」「アイコンの習得性」が手がかりとなる。この規格はアイコンの設計・評価に関する用語や特性を定義したJIS規格である。
  3. JIS X 9303-1で定義されているアイコンの評価特性の種類を確認する。
  4. この規格では、アイコンに関する複数の特性（習得性、判別性など）がそれぞれ別個に定義されており、各選択肢はこれらの特性の説明文の一部を抜き出したような構成になっている。
  5. 「習得性（Learnability）」の定義に着目する。
  6. 習得性とは、ユーザーがアイコンの意味を一度学習・理解した後、時間が経過してから再びそのアイコンを見たときに、どれだけ容易にその機能を思い出すことができるかという、記憶からの想起のしやすさを表す特性である。
  7. 他の選択肢が指す別の特性との違いを確認する。
  8. エの「空間的、時間的又は文脈的に近くに表示された別のアイコンから区別できるか」という記述は、規格上では「判別性（discriminability）」の定義に該当し、習得性とは異なる特性を説明している。
  9. イ・ウについても、それぞれ図柄の区別のしやすさや、既存の経験に基づく識別のしやすさを述べたものであり、習得性（思い出しやすさ）とは異なる観点の記述である。
  10. アの記述「アイコンによって表現されたシステム機能が、それが理解された後に、どれだけ容易に思い出すことができるかを示す」が、習得性の定義（一度理解した内容をどれだけ容易に想起できるか）と完全に一致することを確認する。
  11. 以上より、アを正解と判断する。
