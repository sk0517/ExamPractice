## 問題文

SQL 文に示す参照制約が存在する"商品"表と"受注"表とがある。"商品"表の行を削除したとき，削除した行の商品コードと同じ値の商品コードをもつ"受注"表の行を自動的に削除する SQL 文として，a に入れる字句はどれか。

〔SQL 文〕

```sql
CREATE TABLE 商品 (
    商品コード    CHAR(4) PRIMARY KEY,
    商品名        VARCHAR(64)
)

CREATE TABLE 受注 (
    受注番号      INTEGER PRIMARY KEY,
    商品コード    CHAR(4),
    受注個数      INTEGER,
    納品日        CHAR(8),
    FOREIGN KEY (商品コード) REFERENCES 商品(商品コード)
    ON DELETE   a
)
```

ア　CASCADE　　イ　RESTRICT
ウ　SET DEFAULT　　エ　SET NULL

## 正解

**ア**：CASCADE

## 選択肢補足

| 選択肢 | 内容 | 補足 |
|:--|:--|:--|
| **ア** | **CASCADE** | **正解。親表（商品）の行が削除されたとき、それを参照する子表（受注）の対応する行も連動して自動的に削除する** |
| イ | RESTRICT | 子表に参照している行が存在する場合、親表の行の削除自体を拒否（エラーにする）。自動削除は行われない |
| ウ | SET DEFAULT | 親表の行が削除されたとき、子表の外部キー列の値をあらかじめ定義された既定値に変更する。行の削除は起きない |
| エ | SET NULL | 親表の行が削除されたとき、子表の外部キー列の値をNULLに変更する。行の削除は起きない |

## 解き方

1. 問題文の要求を整理する。
   - 「"商品"表の行を削除したとき，削除した行の商品コードと同じ値の商品コードをもつ"受注"表の行を**自動的に削除する**」とあり、子表（受注）の行そのものを消すことが求められている。
2. ON DELETE 句に指定できる各オプションの動作を確認する。
   - CASCADE：親の削除に連動して子の該当行も削除する。
   - RESTRICT：子に該当行があれば親の削除自体を禁止する。
   - SET DEFAULT：子の外部キー列を既定値に置き換える（行は残る）。
   - SET NULL：子の外部キー列をNULLに置き換える（行は残る）。
3. 「子表の行を自動的に削除する」という要件に合致するのはCASCADEのみである。
   - RESTRICT、SET DEFAULT、SET NULLはいずれも子表の行自体は削除されず、列の値が変わるか、削除がブロックされるだけである。
4. 以上より、正解は**ア（CASCADE）**と判断する。
