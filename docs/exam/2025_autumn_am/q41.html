テクノロジ系 情報セキュリティ JIS Q 27000真正性信頼性情報セキュリティマネジメント用語

問題文

JIS Q 27000:2019（情報セキュリティマネジメントシステム－用語）における真正性及び信頼性に対する定義 a〜d の組みのうち，適切なものはどれか。

〔定義〕
a　意図する行動と結果とが一貫しているという特性
b　エンティティは，それが主張するとおりのものであるという特性
c　認可されたエンティティが要求したときに，アクセス及び使用が可能であるという特性
d　認可されていない個人，エンティティ又はプロセスに対して，情報を使用させず，また，開示しないという特性

　　 真正性 信頼性
ア b a
イ b c
ウ d a
エ d c

正解

ア：真正性＝b，信頼性＝a

選択肢補足

選択肢 真正性 信頼性 補足
ア b a 正解。真正性＝「主張するとおりのもの」、信頼性＝「行動と結果が一貫」の組み合わせが正しい
イ b c 真正性は合っているが、信頼性にcを当てている点が誤り（cは可用性の定義）
ウ d a 信頼性は合っているが、真正性にdを当てている点が誤り（dは機密性の定義）
エ d c 真正性・信頼性とも誤った定義（dは機密性、cは可用性）の組み合わせ

各定義文と対応する用語

定義 内容 対応する用語
a 意図する行動と結果とが一貫しているという特性 信頼性（Reliability）
b エンティティは，それが主張するとおりのものであるという特性 真正性（Authenticity）
c 認可されたエンティティが要求したときに，アクセス及び使用が可能であるという特性 可用性（Availability）
d 認可されていない個人，エンティティ又はプロセスに対して，情報を使用させず，また，開示しないという特性 機密性（Confidentiality）

解き方

  1. 問題文のキーワードを整理する。
  2. 「真正性」「信頼性」というJIS Q 27000:2019特有の用語の定義を、a〜dの4つの定義文から正しく選び出す問題である。
  3. 各定義文のキーワードに着目して、対応する用語を判別する。
  4. 定義a：「行動と結果が一貫している」→ システムが安定して期待どおりに動作するイメージ → 信頼性
  5. 定義b：「主張するとおりのものである」→ 本人確認・なりすましでないことの証明のイメージ → 真正性
  6. 定義c：「要求したときにアクセス・使用が可能」→ 使いたいときに使える状態 → 可用性
  7. 定義d：「使用させず，開示しない」→ 許可されていない者に情報を見せない → 機密性
  8. 真正性と信頼性は名称が似ているため混同しやすいが、次のように覚えると区別しやすい。
  9. 真正性＝「私が私であること（なりすましでないこと）」
  10. 信頼性＝「動作・結果が安定していること（故障や不具合が起きにくいこと）」
  11. 選択肢の組み合わせを確認する。
  12. 真正性＝b，信頼性＝aとなる選択肢はアのみであり、これが正解となる。
  13. 機密性（d）・可用性（c）はCIAの3要素（機密性・完全性・可用性）の一部であり、真正性・信頼性とは別概念であることも併せて押さえておくと、選択肢ウ・エの誤りに気づきやすい。
