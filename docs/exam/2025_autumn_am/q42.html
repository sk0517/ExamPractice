問題文

CSPM（Cloud Security Posture Management）はどれか。

ア　クラウドサービスにおける情報漏えいなどのリスクを低減するために，設定の監視を行ったり，自動的に不備を修正したりするサービス
イ　従業員からの情報セキュリティマネジメントシステムに関する問合せに応じ，必要な支援を行うクラウドサービス
ウ　従業員によるインターネット上のSaaSの利用状況を可視化することによって，ポリシー違反を検出し，接続制限を行うサービス
エ　ファイアウォール，マルウェア検知などのセキュリティ機能を統合し，パッケージとして提供するクラウドサービス

正解

ア：クラウドサービスにおける情報漏えいなどのリスクを低減するために，設定の監視を行ったり，自動的に不備を修正したりするサービス

選択肢補足

選択肢 内容 補足
ア CSPMの説明 正解。クラウド（主にIaaS/PaaS）の設定状況を継続的に監視し，設定ミスやコンプライアンス違反を自動検出・是正するサービス
イ ISMSに関する問合せ支援サービス ヘルプデスク的なサポートサービスの説明であり，CSPMの機能ではない
ウ SaaS利用状況の可視化・ポリシー違反検出 CASB（Cloud Access Security Broker）やSSPM（SaaS Security Posture Management）に近い説明であり，CSPMとは監視対象（SaaS vs IaaS/PaaS）が異なる
エ セキュリティ機能を統合したパッケージサービス UTM（Unified Threat Management）に近い説明であり，CSPMの機能ではない

解き方

  1. 問題文のキーワードを整理する。
  2. 「CSPM（Cloud Security Posture Management）」というクラウドセキュリティ関連の略語の定義を問う問題である。
  3. 略語の意味を分解して考える。
  4. Cloud（クラウド）＋ Security（セキュリティ）＋ Posture（態勢・状態）＋ Management（管理）＝「クラウドのセキュリティ状態を管理する」仕組みであると推測できる。
  5. CSPMの主な特徴を確認する。
  6. CSPMは，IaaSやPaaSなどのクラウド基盤の設定状況を継続的に監視し，設定ミスやコンプライアンス違反といったリスクを自動的に検出し，是正（修正）することを目的としたサービスである。
  7. 他の選択肢との違いを確認する。
  8. イ：問合せ対応・支援サービスであり，設定監視とは無関係。
  9. ウ：SaaSの利用状況の可視化とポリシー違反の検出を行うのはCASBやSSPMの役割であり，CSPMが対象とするIaaS/PaaSとは監視対象が異なる。
  10. エ：複数のセキュリティ機能をパッケージ化したものはUTM（統合脅威管理）であり，CSPMとは異なる概念である。
  11. 「設定の監視」「自動的に不備を修正」というキーワードが完全に一致するアを正解と判断する。
